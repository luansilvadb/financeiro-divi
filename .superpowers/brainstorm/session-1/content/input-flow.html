Como o usuário deve preencher os dados para manter a "baixa fricção"?

A

Wizard / Passo-a-passo

1. Valor -> 2. Quem pagou -> 3. Divisão. Evita sobrecarga cognitiva.

B

Formulário Único (Smart)

Tudo em uma tela. Rápido para usuários experientes. Usa seletores inline.

C

Entrada Rápida (Estilo Chat)

"Pizza 50 reais luan pagou". O sistema interpreta e abre o formulário pré-preenchido.

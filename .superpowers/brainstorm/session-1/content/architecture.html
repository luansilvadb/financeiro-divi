Proposed Architecture: DIVI V1

A high-precision financial orchestrator with a focus on auditability.

System Components
Frontend: Vue 3 + TypeScript (Vite)
  • PWA: Offline support for entry, OCR for receipts.
  • Store: Pinia for state management.
  • UI: Vanilla CSS (SFC style) for custom, polished look.
Backend: Supabase
  • Auth: Email/Password + Social (Google).
  • Database: PostgreSQL (RLS for P2P privacy).
  • Storage: Comprovantes (Receipts) images.
Core Logic: Ledger Service
  • Distinguishes Quem Pagou, Quem Deve, and Beneficiários.
  • Handles hybrid splits (Weights, Equal, Fixed).
  • Immutable logs for every transaction change.

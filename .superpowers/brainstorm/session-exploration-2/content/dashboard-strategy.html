Estratégia: Core Domain (CalculadoraSaldos)

Novo Método: obterExtratoMembro(id, transacoes)

Em vez de o componente calcular tudo, o Service retornará uma lista de objetos ricos:

{
  transacaoId: string,
  descricao: string,
  data: Date,
  valorLiquido: Dinheiro, // (Pagou - Consumiu)
  saldoAcumulado: Dinheiro // (O saldo após esta transação)
}[]
  • Testes Blindados: Podemos testar o cálculo do saldo acumulado com 100% de precisão no Vitest.
  • UI mais burra: O componente apenas faz o loop e exibe os dados.
  • Performance: Centraliza o loop pesado em um único lugar otimizado.
